üìñ **Description**
---------------

Myntra is a modern e-commerce website built using React, a popular JavaScript framework, and Firebase, a cloud-based backend platform. The project aims to provide a seamless online shopping experience for customers, allowing them to browse and purchase products from a wide range of categories. The website features a responsive design, ensuring a smooth user experience across various devices and screen sizes.

With Myntra, customers can create accounts, manage their orders, and access their order history. The website also provides a robust search functionality, making it easy for customers to find their desired products. The website is fully integrated with Firebase, allowing for real-time authentication, secure data storage, and seamless communication between the front-end and back-end.

Myntra is a comprehensive e-commerce solution that combines the power of React and Firebase to provide a fast, secure, and scalable online shopping experience.

üìÅ **Project Structure**
-------------------

* **components**: Contains reusable React components, such as header, footer, and product list.
* **containers**: Contains React containers, such as App, Home, and Profile.
* **utils**: Contains utility functions, such as cn and useToast.
* **store**: Contains the Redux store configuration and reducers.
* **firebase**: Contains Firebase configuration and authentication functions.
* **auth**: Contains authentication functions, such as createUserWithEmailAndPassword and signInWithEmailAndPassword.
* **index.js**: Contains the main application entry point.
* **package.json**: Contains project dependencies and scripts.
